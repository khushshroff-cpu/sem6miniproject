# 🎯 Career Guidance System

## 📌 Overview
This project is an AI-based career guidance system for 10th-grade students.  
It uses aptitude testing and user interest to recommend the best career stream.

## ⚙️ Features
- 30 MCQ Aptitude Questions
- Logical, Numerical, Verbal Analysis
- Machine Learning (Random Forest)
- Interest-based recommendations
- Stream Suggestions:
  - Science
  - Commerce
  - Arts
  - Diploma Engineering
